# FootballBuddy- Football Statistics Platform


A premium football analytics platform delivering live scores, league standings,
team rosters, player bios, and AI-driven match analysis — all without relying
on paid APIs.

## Features
- 🔴 Live match scores and fixtures
- 🏆 League standings, stats, and fixtures across major leagues
  (Premier League, La Liga, Serie A, Bundesliga, Champions League, and more)
- 👥 Team profiles and squad rosters
- 📖 Player biographies via Wikipedia
- 🤖 AI match analyst for tactical insights
- ⚡ Real-time data scraped directly from FotMob

## Tech Stack
- **Backend:** Node.js, Express
- **Frontend:** Vanilla JS, HTML, CSS
- **Data Sources:** FotMob (scraped), Wikipedia REST API
